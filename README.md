### Tech stack

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/hnguy59/bubble-booking.git
    ```
2. Fill out the `.env` variables (an example can be found in `.env.example`)
3. Install NPM packages
    ```sh
    yarn install
    ```
4. Run on your local
   ```sh
   yarn dev
   ```
